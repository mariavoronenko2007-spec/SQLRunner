# SQLRunner

## Описание проекта

SQLRunner — это консольное приложение на языке Python для выполнения SQL-запросов к базе данных PostgreSQL.

Программа принимает SQL-запрос от пользователя, проверяет, что это SELECT-запрос, при необходимости автоматически добавляет LIMIT 5, выполняет запрос и выводит результат в виде таблицы.

---

## Возможности программы

- подключение к базе данных PostgreSQL;
- выполнение только SELECT-запросов;
- автоматическое добавление LIMIT 5, если он отсутствует;
- вывод результатов в виде таблицы;
- обработка ошибок.

## Необходимые библиотеки

Для работы программы необходимо установить библиотеки:

```bash
pip install -r requirements.txt
```

---

## Настройка

Перед запуском необходимо создать файл `.env` в папке проекта.

Пример содержимого файла:

```text
DB_HOST=localhost
DB_PORT=5432
DB_NAME=school
DB_USER=postgres
DB_PASSWORD=your_password
```

Вместо `your_password` необходимо указать пароль от вашей базы данных PostgreSQL.

---

## Запуск программы

Откройте терминал в папке проекта и выполните команду:

```bash
python main.py
```

---

## Пример работы программы

После запуска программа попросит ввести SQL-запрос.

Пример:

```sql
SELECT * FROM students
```

Если в запросе отсутствует `LIMIT`, программа автоматически добавит:

```sql
LIMIT 5
```

После выполнения запроса результат будет выведен в виде таблицы.

---

## Ограничения

Программа выполняет только SELECT-запросы.

Запросы INSERT, UPDATE, DELETE, CREATE, DROP и другие не выполняются.
